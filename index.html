UTSelva - Sistema de Información Institucional Administrativa

Dentro de la Universidad Tecnológica de la Selva fui parte del desarrollo de un sistema para el control de administración de ingresos, egresos, dsitribución, apoyos y proyectos (POA, PROY, PIDE, REQ, META), etc. Este sistema serviría para mantener en orden todo tipo de facturación en cuestion economica.
Durante los 8 meses de desarrollo trabajé como FullStack, en el FrontEnd como en el BackEnd, además de creación de tablas SQL, consultas con filtros, con Joins para crear consultar que exigir al servidor y que la respuesta fuera rápida y eficiente.
Las tecnologías aplicadas a este proyecto, fueron:
Html 5, Css y JavaScript como FrontEnd, Python (FastAPI) como BackEnd, SQLlite como SGBD 1ra version, PostgresQL como SGBD permanente, Tabler.io y Bootstrap como librería de estilos, JinjaTemplates para crear plantillas reusables, Jquery para creación de funciones y creación de peticiones al servidor, TortoiseGit y Git para el control de versionamiento y por último KanBoard para la asignación y creación de tareas.

Plataforma IoT para control de acceso a Centros de Datos ECOSUR

Dentro de la empresa, fui parte del equipo de la unidad de Tecnologías de la Información y Comunicación, desarrollando un sistema de control de acceso remoto para una puerta inteligente, el proyecto estaba dirigido a establecer una bitácora de control de acceso físico al centro de datos y a los activos de información esenciales, describiendo la actividad a realizar en estos y restringir el acceso físico a personas internas o externas, y permitirlo únicamente con autorización escrita y los registros correspondientes, a través de una plataforma integral basada en tecnología IoT.
Durante los 4 meses de trabajo y desarrollo, se logró tener una versión beta del sistema, además de su despliegue para funcionamiento y pruebas necesarias.
Las tecnologías implementadas dentro del sistema de control fueron:
Laravel como BackEnd, React JS como FrontEnd, HeidySQL como SGBD, Tabler.io como librería de estilos, CSS como hoja de estilos y Material UI para la creación y estilos de componentes.
